## 談談我眼中的新疆:兼聽則明，有別於部分“牆外”的聲音
`2020-03-23 08:24 GM37` [轉載自GNews](https://gnews.org/zh-hant/149575/)

作者：胡楊

2009年7月6日的子夜，當我在大都市的車流中一邊聽著邁克傑克遜，一邊欣賞夜景時，一通來電讓我不得不在慌亂中靠邊停車，回撥過去… …

你跟我仔細說清楚，新疆到底怎麼了？

“我也是才看到，新疆那裡暴亂了，死了好多人！”

哪裡？南疆麼？南疆過幾年就會來那麼一下，問題不大，別大驚小怪的。

“不是不是，這次不一樣，這次在烏魯木齊，特別兇！你不信你問問那邊的人。維族在街上看到漢人就殺！”

我問問去，應該沒事。烏魯木齊以前也有公交車爆炸。

… …

翻開通信錄，找到新疆的朋友。開始一個一個的打過去，竟然全部無法接通！這麼多年來，除了08年汶川地震發生時，聯繫災區附近的朋友出現過這種情況，但那也僅僅是過了一天就恢復了。一切都會沒事的，明天再聯繫聯繫，可能就是不讓暴亂的人再相互聯繫。

7月7日，午飯後，在茶桌旁準備聽聽新疆當地的朋友怎麼跟我描述這件事。一個都沒打通，竟然一個都沒打通。開始聯繫不在本地的新疆朋友，竟然也都是一頭霧水。彼此相互寬慰之後，約定有了新的信息相互溝通。

7月8日，就這樣又過了一天。在和同事的聊天中，有人提醒我：笨呀！你“翻牆”出去看看不就都知道了。是啊，我真的好笨！我竟然“忘”了我能查到的信息，都是在一個局域網內的信息，都是有“人”想讓你看到的你才能看到！

7月9日，自己在家裡折騰到半夜，“翻牆”出去，搜索開始，滿屏的信息都在介紹“75事件”。圖片、文字、視頻… …我努力的辨識這些信息所記錄的地點，無法想像這居然發生在烏魯木齊，這裡面的景象居然就像我在“CCTV”裡面看到的阿富汗街頭！有的鏡頭記錄我看了幾十秒就看不下去了。我腦中的第一反應就是：這是打仗了麼？維族造反了？這還是我熟悉的新疆麼？這是要“獨立”了麼？那麼多駐軍，還有新疆建設兵團呢？這麼多鎮壓暴亂的機構怎麼能允許發生這種事情呢？漢人不會反抗麼？這是民族仇殺還是不滿政府？ … … 我不敢看下去，我更不願意看上面的留言評論！因為在當時我認為在國外的所有人對我的國家是帶著“有色眼鏡”的！是不負責任的！是故意想要分裂中國的！是“亡我之心不死”的！是蓄意顛覆無產階級專政的！是壞人！

刪掉VPN，我等著了解“真實”的信息，就這樣在電視節目中了解屬於我“應該”知道的資訊。不知過去了多久，我聯繫到了新疆的朋友，他在電話裡告訴我：一切都好，你認識的人也都好，不用操心，就是上網不方便了，過陣子再說吧。

10年過去了，我慶幸我不是“75事件”的親歷人，也慶幸我牽掛的人沒有在這個事件中受到傷害。可在這片土地曾生活過幾十年中的“偶遇”經歷，現在回想起來也心有

餘悸。曾親眼目睹過的、以及身邊朋關於這裡林林總總的遭遇，一直都想以自己的角度向朋友們介紹一個我眼中的新疆。

新疆，除了她被賦予美麗、遼闊、熱情等等一切都不為過的讚譽之外，相信在不少人的心裡，還有一種映像就是“神秘”！然而羅布泊、古樓蘭、唐王城、阿勒泰山下的大石堆等等不是我此文的內容，我描述的“神秘”是很多朋友搞不懂的“省中省”——新疆生產建設兵團。
![](https://s3-ap-northeast-1.amazonaws.com/news.guo.offload.media/wp-content/uploads/2020/03/23080923/1_%E5%89%AF%E6%9C%AC.jpg)
這是一個集“軍、政、企”功能三合一的副省級組織。兵團“最高”長官——兵團黨委書記、政委，同時任自治區黨委副書記。自治區黨委書記同時任兵團“第一政委”。這強調了新疆生產建設兵團歸屬於自治區領導管轄。同理，自治區所屬的伊犁州、博爾塔拉州、巴音郭楞州等州黨委書記也同時任職兵團某師的第一政委。

新疆生產建設兵團的由來和發展歷程，我在這裡就不多贅述了。我認為需要講清楚的就是關於兵團下屬各師與市的關係。舉個典型的的例子：以新疆生產建設兵團第二師為例，距離巴音郭楞州的庫爾勒市50公里處，有一個鐵門關市，這裡就是第二師的師部所在地。師政委同時任鐵門關市的書記，因為鐵門關市的規格是縣級市，但因為二師的緣故，因此這裡的最高領導是正廳級。很多朋友搞不明白為什麼一個縣里面的局長是處級，就是因為這個緣故。因為這裡是“師、市合一”的組織編制。下屬各單位是“團”，團部所在地，現在基本都為“鎮”。比如兵團第一師下屬的五團，駐地就是“沙河鎮”，也是新疆的特色小鎮之一。

綜上，因為有新疆生產建設兵團的存在，所以新疆一些地區的行政編制使得外人經常摸不清頭腦，也是因為有“兵團”這個編制在新疆的建立、發展，給新疆的各族人民帶來了各種各樣的“變化”。

新疆生產建設兵團，是在新疆“和平解放”之後，為了維護新疆的“穩定”，加強新疆的“統治”所設立的一個特殊編制。這裡的人來自全國五湖四海，從文化、家庭等各方面背景都是天差萬別，有教師、技術工人，也有逃荒者。但由於長相、語言、生活習慣等差別，這些進疆的漢族天然就與本地的“少數民族”形成了一定的“距離”。然而在更久遠的年代就來到新疆定居的漢人卻與本地的民族同胞保持著更親密的關係。

在北疆東部的哈密、吐魯番，距離烏魯木齊100公里左右的瑪納斯、沙灣等地域，世居在此的漢族人會習慣的稱呼少數民族為“朋友”；而兵團的各地漢族民眾，則多稱呼少數民族為“老鄉”，同時也被少數民族稱為“幹部”。一個簡單的稱謂習慣，從一個側面也可以反應出一種相互的距離。

新疆生產建設兵團，在經歷了“文革浩劫”後期（1975年）被撤銷改制為“農墾局”，又在1981年年底恢復了編制。旨在“屯墾戍邊”，“防禦霸權主義侵略”等所謂的“國家戰略”。隨之逐步的開荒、種植、發展經濟。至今為止，監製包括了14個師、185個團。為了避免國際上對於新疆生產建設兵團這個體制“兵團”的理解，以及適應在經濟活動中的便利，1998年，兵團組建法人單位“中國新建集團”，對外合作均適用該名稱。

在兵團的建成、裁撤，再建成的跌宕歲月中，來自全國各地的“兵團人”承受著各種政策變化所帶來的“爹不親，娘不愛”的尷尬局面。因為自身沒有稅收，甚至沒有地。因為稅收都是所在地區的“自治區編制單位”收取後，上繳自治區，再通過自治區財政進行各項統一的安排下撥。例如：兵團重點石河子市，它的稅收，根據屬地劃分，被周邊的瑪納斯縣，沙灣縣分別收取。

這樣一個“怪胎”的建制，在中央不同的領導執政期間，都對其有著千差萬別的影響。近些年，建設兵團又經歷了一次差點被裁撤的風波。中共十八大之後，負責“統戰工作”的俞正聲調研新疆，回京後，在常委會議上，將新疆生產建設兵團批的一無是處，幾乎可以否定了兵團這個單位對於共產黨“維穩新疆”、“屯墾建設”的所有貢獻。該講話原稿根據指示下發到兵團“團一級”單位（正局級）學習，可笑的是，該文件在傳達至各級單位之後甚至有的還不到12個小時，就責令各地在最短時間內，將所有傳閱文件第一時間收回並在監督下銷毀。文件雖然收回了，但講話的內容，也刺痛了兵團人的內心。屯墾三代、四代人，在隔壁中開荒種地，建設城鎮。除了微薄的工資還算穩定之外，子女上大學可以填報的學校和專業方向以及招生人數相比地方（自治區）少得可憐。看似很多定向招生的專業大多就是養殖、農業、畜牧。在兵團人的經歷中，唯一值得驕傲的，可能就是在大饑荒中，這里基本保障了基礎糧食的供給，也因此在那個年代，很多內地的親戚投靠來新疆兵團，免遭餓死的厄運。

在2017年，陳全國來疆主政，隨著高壓維穩的政策推行，兵團“維穩中堅力量”的功能又再次被提及，同時，加強了新疆兵團在“南疆四地州”的“計劃發展”。新建城鎮、團場，引入人口，招商引資等等政策又一步一步的推行開來。在新疆自治區的幾次會議中，兵團“向南挺進”的提法，不斷的出現在文件中。從北疆地區的兵團幾個發展較好的師市，抽調各級單位的人員對南疆各師進行“援助”，行政級別均上調半級。與此同時，為了進一步加強自治區對於兵團的控制和“改革”，又搞起了“幹部兵地交流”的活動，地方上的人員來兵團任職，上調半級，兵團人員到地方，平級或下調半級。各種所謂的“不平等待遇”也讓兵團人深感糟心。所謂的“到南疆第三次創業”的話題在兵團人口中就如同“夜壺理論”一般被詬病。

兵團人絕大多數都想不通，自己的命運為何就像自己所在的位置一樣。遙看四周，風高雲淡，大有任鳥飛的豪情；端詳自身，大漠深處，何時方能見綠洲的絕望。是農民？是職工？是軍人？亦或什麼都不是……

恐怕將子女送出新疆，已經是絕大多數兵團人此生的奮鬥目標了。不知道他們是否還會回想，在那個“山東痞子”王樂泉主政新疆之時，兵團“不務正業，吃拿卡要，貪圖享樂”的“好生活”呢？

在王樂泉“統治”新疆十五年還久這期間，新疆的發展，基本就是沒有發展，除了能給他本人和利益集團帶來好處的行業企業之外，一切的市場經濟活動，都可以說是一種越來越滯後、越來越無序的年代。

在這個集權政府尤為突出的新疆，甚至於很多內地人，都有一句“社會主義在新疆”的調侃。貧窮，計劃，集中，權利，等級… …王樂泉在新疆主政時期，無論是地方，還是兵團，行政官員對於權利行使的“充分”，私慾的極度膨脹，已經完全忘記了這片土地上生存的各族人民。為了自己利益的滿足，藉著兵團等各大體制單位的種種便利，肆意的侵占資源、踐踏環境。至於民生、民權？呵呵… …

全國的“天價切糕”、“維吾爾小偷” 、“南疆暴亂”、“烏魯木齊公交車爆炸” 等等惡性的事件不斷在給新疆抹黑。可絕大多數的人，僅僅看到了表面，因為民族、宗教，以及共產黨的宣傳管制，這些問題幾乎很難被完整的報導。但是新疆本地，特別是南疆的人們大多都知道，因為政府的無能和貪婪，將各民族人民壓榨到極致，尤其是淳朴善良的民族同胞。各種困難沒有人理會，應有的保障沒有人兌現，在這種情況下，一個隱形的勢力，悄悄的滲透到了各家各戶之中，他們用“傳教”的方式欺騙，用金錢來引誘，用恐怖來威脅。帶走了他們的子女，甚至更多的是兒童。隨後有組織的派發到全國各地，偷、搶、賣淫、販毒等等的犯罪活動為組織者帶來了豐厚的利潤，同時又通過一些外表合法的商舖、餐館來為這些非法活動做了掩蓋。

同時，他們仍不滿足於此，開始成批的組織各家的兒童，送往境外的恐怖組織營地進行洗腦和培訓。他們非但不告知造成這一切苦難的根本原因在於這個政黨，而是將這一切歸咎於所謂的“漢人”欺占我們的土地，掠奪了我們的資源，殘害了我們的同族。這似乎與本地的漢族，特別是兵團人接受到的宣傳教育恰恰相反。每一個兵團人都“知道”，身上肩負了鎮壓“少數民族暴動”的“神聖使命”。 1998年的南疆暴亂被平息後，幾乎所有的兵團人，哪怕是中學生，都被有組織的參加主題教育和展覽。裡面的血腥場面讓每一個人都難以釋懷。但是，誰又曾想起，在90年代下崗潮、待業潮的時代，一個小小的城市石河子，竟然失業率、犯罪率在全國都名列前茅。

在那十幾年，暴亂時有發生；烏魯木齊、和田、喀什等這些大一點城市的治安環境也不好。我本人在為數不多的幾次烏魯木齊的行程中就被偷過甚至搶過。這與內地相比唯一的“特徵”就是實施者基本是維族。王樂泉在新疆，一方面利用他“封疆大吏”的特權，狂攬財富。新疆本地人都戲稱他為“山東人民的好兒子，新疆人民的賊娃子”。從基建工程到農業生產，他的家族和“鄉親”無孔不入。比如新疆的首富“孫廣信”和他的新疆廣匯集團與王樂泉的聯繫，就是當地老百姓茶餘飯後的“談資”。另一方面，無論是自治區或新疆生產建設兵團的職工，收入等待遇卻不曾得到有效改善甚至停滯。本地大多數人都認為這是他為了向“北京”邀功：新疆人民在他的管理下基本生活已經過得很好了，不需要國家這方面的補助。

然而，新疆的真實情況卻是物價高，收入低。在內地經常聽到“新疆的錢好賺”，我個人認為，一方面是因為新疆的各級官員知識層次低，特別是新疆生產建設兵團的官員“膽子大，敢伸手，敢辦事”；另一方面是因為這裡距離內地遙遠，各種新興事物，新疆接觸的晚，因此，雖然人口不多，但也提供了足夠的獲利空間。

在“75事件”發生之前，少數民族可以帶著“英吉沙刀”上街，可以對著漢族人“耀武揚威”，甚至可以“明搶”。需要聲明的一點是，我只是描述一種社會狀態，並不是對於少數民族有任何的仇視。因為我也有很多少數民族的好兄弟！烏魯木齊的市民上街喜歡把雙肩包背在前面；習慣時不時回頭看一下；公交車上的提示語不是防賊而是留心座位下的可疑包裹；漢人在大巴扎（集市）如果對著一個攤位多看兩眼就基本必須要買一些… … 我想還有很多諸如此類的不好回憶都會在每一個新疆的漢人腦子裡。

雖然在世界各地，民族問題都是比較複雜和棘手的問題之一，但是在新疆，民族問題的本質，卻是因為執政者一手造成！新疆豐富的各種物產資源（礦產、石油天然氣等）被源源不斷的輸入內地，卻絲毫沒有因此獲得更好的生活；從內地來新疆工作的兵團職工，背井離鄉卻沒有得到應有的待遇；教育、醫療等社會公共資源的嚴重落後，卻沒有得到有效的扶持提升。基層幹部把本已經少得可憐的各項農業補助攬入囊中，讓包括漢族在內的各民族老百姓生活在貧困交錯之中。讓少部分犯罪分子有機可乘，販賣少數民族兒童到內地進行盜竊活動；“只要無關乎自己的既得利益就放縱不管”的政策也讓恐怖極端思想滲入到少數民族之中；“新疆王”的作風讓政治生態更進一步的惡化；官員的“集體作惡”，無能的“民族政策”，不僅引發了少數民族民眾抗議，同時也讓漢族民眾心生不滿，並且這種情緒更進一步帶來了各民族之間的隔閡與對立。更有產生民族仇恨的傾向。

“75事件”發生後，烏魯木齊的市民上街遊行，高喊：“讓王樂泉滾出新疆去”。當他調任北京時，甚至有人放鞭炮慶祝。張春賢來疆任職期間，雖然在烏魯木齊市進行了BRT建設、高架橋建設、地鐵建設這些“民生工程”，雖然以一副“親民團結”的形象展示，但是並沒有能夠在根本的各項政策上做出有效的調整。

2016年，隨著陳全國的到任，新疆開始發生了一系列的變化。

首先，自治區以及生產建設兵團的官場地震開始了。隨著“維穩壓倒一切”的逐級推進，各級官員開始了“官不聊生”的新生活。 “實現三不出”作為“維穩”工作的終極目標（不出大事、不出中事、不出小事）。各行各業都領到了更加細化的具體任務。公務員、企事業單位、甚至商舖都要配合這個目標。黨員幹部要和少數民族結為“親戚”，要定時去少數民族家裡“同吃、同住、同勞動、同學習”，並且開發了用於監督此項活動的APP（地點大卡，活動記錄，照片上傳）；

各種各樣的“制度”五花八門，事無鉅細。出一冊書都不為過！牆外的人大多都在討論“集中營”非法關押了多少少數民族，可你們是否知道，在“高壓維穩”的政策下，還有很多官員因為檢查時候不在“親戚”家；因為值班時間不在辦公室；因為微信發了牢騷抱怨；因為被人“點炮” 等等都會被扣上“維穩不力”的帽子被關進去剃光頭吃牢飯。甚至“高壓維穩”成了某些人清理“政敵”的有效可行手段。

體制內的基層人員需要自掏腰包買米麵油去“親戚”家，單身女性漢族要和單身男性“親戚”同吃同住在一個屋簷下等等這些看似荒唐的事情數不勝數。北疆的干部被抽調去南疆任職三年；抽調幹部去“駐社區”，“駐村”，“駐寺”一到三年不等；從內地招聘來的“輔警”全年無休，不能辭職，工資卻不能發全；招聘來的大學生幹部，有的干了半年就算不要檔案也要辭職；各級政府“引人入疆”卻大多是引來了低端的農民和“戶口本”… …

體制內的人一年到頭沒有幾天安生日子。來新疆做生意的內地人紛紛跑了回去。維族人難過，漢族人也難過。在這場“運動”中沒有贏家！

我們譴責無人道的“集中營”，但是請不要忘記在暴亂中死傷的各族人民和他們的親人！這幾年來的確是沒有再發生任何暴恐事件，但是新疆付出的代價太大了！難道就真的沒有第二種方式麼？難道把人關在“集中營”就真的能夠讓他們“棄惡從善”麼？更何況絕大多數人都是所謂“潛在危險分子”。我的一些維吾爾族兄弟跟我說：“我們更恨極端恐怖主義，想辦法掙點錢過好日子難道不好麼？”我想，他們肯定會迷惑，為什麼遭受突如其來的災難，更不會明白，這一切的根源是什麼！但是，如果思想上的“問題”都可以入獄，那麼誰能保證思想不反复呢？也許“誅心”的本事，從歷史上看，已經手到擒來，爐火純青啊！

曾有一次機會，被“幹部”邀請參觀“集中營”，可我婉言拒絕了，僅僅是繞著周圍走了一圈。回想起那一刻，我感到了自己的懦弱。我不敢直面甚至不敢想像那裡發生的境況。如果連司法的程序都沒有履行，就可以把人送到這佈滿鐵絲網、電網、攝像頭的院子裡，就可以讓人在“保安”、“民警”、“武警” 的“緊密照料”下度過每一天。那麼進去的人會遭受到什麼樣的待遇真是可想而知。特別是在這個可以曾在北京大規模槍殺進步學生的罪惡政權的統治下，還有什麼是不可能發生的呢？

新疆及兵團的監獄管理系統，紀檢委系統、公安系統，這短短幾年都爆出塌方式腐敗的“窩案”。一個紀檢委的女官員，可以在“談話室”肆意的對“被談話人”連續扇幾十個耳光，可以拿高跟鞋砸人，踩人！一個女性，對自己體系內“同志”都可以做出如此喪心病狂的舉動；一名局級的警察，可以從毒販中挑選一個做自己的“老婆”，一起雙雙“賽神仙”，甚至以罰養吸；一名監獄系統高官，在自己的體系下任何一名獄警的調動、升遷，都必須要他自己簽字批准，狂攬不義之財！一個國有地方壟斷企業，虧損到市長一周去開三次專項會議，然而卻任然在自己的大樓裡各項奢靡！

烏魯木齊市政府，因為同時推進多條地鐵線的資金出現問題，同時因為維穩的壓力，竟然在會議上提出，把已經挖開的地鐵隧道進行回填，他們真的以為挖土需要資金，填土就不需要了麼？另一個縣的政府，提出要把城市後面的荒山通過3年中滿樹，政府沒錢就強迫市屬企業拿錢，當企業問到這個項目沒法收回投資，政府官員回答到：把山圈起來賣門票嘛！難道他轄內的老百姓就真的那麼稀罕遠離城區十幾公里的山樹麼？ ！

這樣的例子比比皆是。難道這還算是個案麼？如果每一個人，都把這些現狀，看作一種習以為常的“常態”，是自己蒙蔽了雙眼，還是蒙住了心？如果認為知識、法律這些基本公民素質都缺失的社會屬於正常，那是不是太過於麻木和自我欺騙了麼？這僅僅是在地廣人稀的新疆，放眼全國，這些“新聞”不是數不勝數麼？ ！難道每個人都更願意關注娛樂明星今天結婚明天離的八卦麼？ ！
