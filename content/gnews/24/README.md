嗩吶、二胡是中國的民族樂器精神上的兩個柱子，嗩吶，二胡。嗩吶吹的是前世，吹的是未來；二胡拉的是過去，這不得了了。都是從西域過來的，沒錯。但是這就是中國文化的了不起了，中國文化能把這些東西漢化以後，中國化以後真的是做的太棒了。

（讀戰友留言：「千年琵琶，萬年箏，一把二胡拉一生。嗩吶一響全劇終，初吻不知曲中意，再聽已是棺中人。黃泉路上人消沉，望鄉台上憶前塵。孟婆一湯忘今生，奈何橋上渡殘魂，來生再做華夏人。 」這有點忽悠。）

我昨天給美國朋友聽聽，他特別震撼。我說我想跟你解釋一個事情，中國人和共產黨的區別在哪裡，這種音樂是不是共產黨創造的？他說那絕對不是。我說今天中國人所聽的音樂，中國人用的文字，中國人做的飲食是不是共產黨創造的？他說不是。我說一切都聽黨的，一切都是黨的，爹親娘親不如黨親，黨給了我們一切。這一切里包不包括音樂、飲食、建築、服飾，語言？他說不包括。

我說這是中國民族的靈魂吧，這是中國幾千年老祖宗給延承下來的吧，在這幾千年面前共產黨算個屁啊。但為什麼共產黨要把這一切全部閹割掉，所有能毀掉的建築全毀了，把語言改成簡體字了，然後中國人現在不會說人話，只會罵人。你看看真正的中國人大街上聊天，我記得最清楚當年，1984年我跟一個台灣的，那時候台灣人很時髦。84年，我跟我們老家范縣，跟一個台灣人聊天。那時候看到一個台灣人就跟看到一個從外星來的人一樣。

他講了幾句話我到現在都終生難忘，所以你一生見什麼人，聽什麼人說話真的太重要了。他就問我，他說：老七啊，你那麼小我請問你，為什麼大陸的女人說話張嘴就是罵人，說話就很沖，好像每個人都生氣一樣，就像我說的，每個人都拿著打火機到處找汽油的感覺。男的一說話一張嘴就是罵人，一說話就是擦擦擦。在北京就是一說話就是「你媽的，你媽的。」成了叫國罵。你見過一個國家、一個民族，一個整個民族女的出口就罵人，男的出口就帶操。你說這是什麼情況？當時這個台灣人簡單的話影響了我，所以說對我的人生影響很大。

共產黨到了中國，一個外來黨，叫做買爹主義，到了中國就給中國人帶來什麼？女的不像女的像男的了，出口全帶刀子，要麼就是噴屎啊，這多可怕，怎麼教好孩子。我們中國最偉大的是女性最偉大，但是共產黨把女性都變了。共產黨當官的，回來了僑辦的一見面，頭一天還客氣，第二天一喝酒，這女的大腿一撇，張嘴就罵。他說國民黨你罵唄，你罵的也太難聽了。口中無德呀！

一個人一個社會基本的道德沒有了，道德其中一條就是你不能嘴巴一張開生殖器就來了，大腿一撇你想罵誰罵誰。人家台灣當時已經是文明啟蒙了，除了當時帶走的精英，人家台灣已經到那個份上了。這是對我一個刺激。

1983年，我到廣東去販賣電子錶、牛仔褲。廣東人說話是很粗魯的，那個聲音也粗魯，港人說話就跟廣東人（差不多），說實在話是很難聽的。但是有一樣讓我也是震撼的，那時候我最多就買個鄧麗君磁帶，買完磁帶我回去還要買個雙卡錄音機，我要複製，複製一盤賣多少錢，我去賣。我弄了兩個（雙卡錄音機），二十四小時給我複製，我還買了空白的帶子。一個我的幾次（生意往來）的客戶，一個香港人，但是見著真香港人很難的，都是假香港人，都是廣東的，就穿著禮帽穿著白褲子，穿個港衫，就是香港人了？這都是裝的。真的香港人是很少的，倒賣那個小磁帶。

但這就是真的香港人，說我聽說一個小孩很講義氣，賒的貨很快就能把錢還上，很不簡單。我要見他會會這個人，這個人跟我現在還保持著聯繫，還活著呢，後來也成了個大富豪。最近有點親共這個哥們有點問題，這麼多年的老友，幾十年了。

他說了一句非常關鍵的話，他說：老七啊，為什麼咱們同胞見面就不打個招呼？沒有人大膽地說：你好，你好，哪有這個，最最簡單的，在我老家：喝湯了嗎？吃了嗎？喝湯了嗎？就你吃飯了嗎，這就不錯了。見老人了，撓撓腦袋（就算）出來打招呼了，是吧？他說為啥不能說：你好，就不能這樣打招呼嗎，因為我一見人面就立刻打招呼：你好，你好，你好。

那時候我到那個深圳去，那時候深圳像個豬圈似的，沒多少人，路上都擺著賣水果的，還有賣走私貨的。那時候羅湖賓館剛剛開始建。我見人就打招呼，人家就覺得很奇怪，說：你這小夥子不錯，小郭。他說：你以後我這貨你不用先付錢，你先賒賬。那時候是十塊錢抓一把電子錶，我一抓，一般人都想抓十把，但是最多能抓五六把，我一抓都能夠抓五六把，甚至抓掉十把，我就那麼能，所以就賺了很多很多錢。他說：老郭你以後不用抓了，10塊錢你拿十把，拿十塊表，電子錶。電子錶多牛啊，那傢伙，都可能娶個媳婦你知道嗎？那不是開玩笑的，真的是這樣子的。牛仔褲，他說：你以後也不用說了，以後你一拿就一打，一打十二條，你就是這個錢，你拿一打就完了，港商，一打，全都是一打，所以說當時讓我賺很多錢。

所以說整個當時那個文化，我就突然意識到，香港和大陸之間缺的就是這種文明，台灣和大陸之間缺的就是對人的尊重。而且為啥人家都聽鄧麗君，人家都問大陸為啥都愛聽鄧麗君的音樂，後來還買那個張帝問答，哎呀我滴媽，那個張帝問答火得不行，我走私那個磚頭的錄音機回去，五倍十倍地賺。你想想我那時候逐漸就明白：人家文明，人家知道中國人的這個歷史上的這個音樂，人家懂中國。

那個時候我就研究為啥鄧麗君的歌那麼好聽。後來這個有意思，我第一次出國我就遇到一高人，我說為啥鄧麗君的歌（這麼火），誰給她寫的，誰給她編的？他說我告訴你鄧麗君的歌都是抄的日本的歌，我一下子就醒悟過來。然後鄧麗君的每首歌我都找日本的原歌聽，完全一樣，都是抄的，到現在台灣的樂壇都抄日本的歌。

那我又問為啥日本人有音樂啊，中國人都沒有音樂嗎。結果人家說共產黨不讓你歷史傳承，就不讓你真正的傳承祖業，共產黨不讓你知道過去的好東西，不讓你知道這一切的東西。他說文貴你這叫啥嗎？這叫做文化壟斷，文化閹割。說得我渾身發熱，我說：這詞好啊，這詞有意思。

從那以後我腦子就有幾根弦。台灣這位歸國華僑回來說的，這那時候他回來多不容易，那時候回來比登天都難。那時候多榮幸啊，就是大陸咱們完全不會說人話。然後香港人告訴我，說咱們不尊重人，不會打招呼、沒有禮貌。然後知道台灣的音樂是抄日本的，日本二戰以後反而延承了歷史最美好的部分。共產黨不讓我們知道最美好的東西，這對於一個13、14歲的我來講那是震撼的影響。

所以說，昨天我跟我美國朋友，我讓他們聽了三段音樂，一個是嗩吶；第二個東方紅，太陽升，中國出了個毛賊東，我還找到了英文版的，他們聽了大笑；接著第三個聽，一個我從小十幾歲到現在都聽的歌曲，就是小螞蟻。小螞蟻是抄的人家的歌，也是西方的歌，我說這就是中國人整個文化和歷史徹底被閹割。

（關於）男人我說：中國的明星，我讓他們看了中國的明星都小男孩，都是奶粉飾的，白到了一種沒有力量、沒有生命的臉，瘦得整個人，中國的男演員你見一個是一個，瘦得（就像）你看到吸大煙的，吸毒的不健康，我從沒見一個明星就是生活中非常棒。

當時我難受，我雖然小，但看的老電影，我這70年出生的，那時候朱時茂和那個叢珊演的《牧馬人》，郭撇子，然後男的喝酒，女的喝茶，給他找的這個叢珊演的女人李秀芝，我說這幾個人我都得認識，我全都得認識。當時流行的牆上掛著中國人明星陳冲啊，這個劉曉慶啊，還有這個叢珊啊，10個明星，我那時候說這十個明星都得跟我睡一遍，那時候我是沒有太太的，結果後來都認識了，都打過交道，但是沒跟人睡過覺，睡覺這是胡扯的。

我認識叢珊以後，感覺叢珊是真的有文化的，叢珊後來嫁給了一個法國人，後來法國離婚又嫁了一個在法國的華人，還有個孩子，我跟他家人都很熟。老茂就不用說了，這是我老鄉。郭撇子，我都認識，謝晉我連他的兒子都認識，他兒子是扯淡的，他爸是十分特別好的人，當時我看了的所有的人我都認識。

《牧馬人》當時候火，拍出來火的時候，我回去問我家人，問我大伯：這電影為啥這麼火，啥意思啊這是？我大伯說這是給我們平反的，給你爹平反的，我說平反啥？他說你爹和我都是被打倒的人，現在是要給咱平反，說你爹的公司就是因為這個電影給平反的。我說電影咋能平反呢。他說電影還有這東西都是共產黨的喉舌，提前放風了。我就感興趣，我說電影還有這功能啊，問問怎麼回事，見人就問，後來我明白還有宣傳機器，宣傳機器。我當時也是讓我很感觸的事情。

我在北京見一高人，這個人後來帶我認識了陳佩斯和陳佩斯他爹，陳佩斯他爹第一個廣告是就給我拍的。這個天梭火腿，開封的，還有當時的這個洛陽火腿，就我拍的，陳佩斯他爹，第一個就是我拍的。那個時候你看看，介紹我認識的就是陳佩斯他爹還有他媽媽，他媽媽跟這個老鄧非常不合，一頓飯就是罵老鄧家，那時候在新華社，在老新華社我記得非常清楚。

但是在那一頓認識午餐和晚餐以及後來的打招呼，我就要搞明白一件事，電影演員，宣傳機器是咋一回事。陳佩斯他爹，老人家真厲害啊，告訴我什麼叫做宣傳機器，說我是怎麼演壞人演到現在的，說現在很多人吐我吐沫，他把我也當成壞人了。老人家說了很多，什麼叫做宣傳機器，什麼叫做演員，說共產黨為什麼要這麼說。哇，啟智，啟智，後來他這個兒子陳佩斯還是不錯的。
<br>所以說戰友們……
