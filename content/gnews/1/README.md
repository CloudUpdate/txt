## 神州租车停牌 中共公司开启多米诺骨牌式坍塌？
`2020-04-02 23:54 GM09` [轉載自GNews](https://gnews.org/zh-hant/160594/)

作者：功夫熊猫

闪崩！香港时间4月3日 10：15 AM港股神州租车停牌，此前一度跌超70％，市值蒸发超60亿 与瑞幸同一大股东。

[瑞幸（Luckin）因财务销售数据造假被披露在美东4月2日一度跌超80%，收盘收跌75.57%](https://gnews.org/zh-hans/160164/)。

有资料表明，陆正耀家族手持集团6.31亿股，占总股本29.77%为第一大股东和实际控制人，联想控股旗下的君联资本、华平投资分别为二、三大股东。 陆正耀同时为瑞幸咖啡大股东。此次被归咎责任的Luckin（瑞幸）的首席运营官兼董事长刘剑，与陆正耀关系匪浅。据瑞幸官网的介绍，刘剑毕业于中央财经大学，在2008年至2015年，先后担任神州租车车辆管理中心副主任和收益管理负责人；2015年至2018年担任神州优车收益管理负责人；2018年5月起担任瑞幸咖啡COO，2019年2月起任董事。

根据19年初路透社的报道称，陆正耀出资并担任董事长的瑞幸咖啡，计划以30亿美元估值在今年五月IPO。同时，陆正耀向高盛、摩根士丹利等投行寻求两亿美元的贷款，以瑞幸咖啡的股票为担保，若瑞幸上市不成功，则以陆正耀持有的神州租车股票为担保。
